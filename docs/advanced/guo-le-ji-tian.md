## 練習筆記

- 和弦非常豐富，有 Gaug、Cm、Bbm7、Bbdim7 等罕見和弦
- 原曲最後有升半音（Eb 調），可以不升直接重複副歌
- Gaug = 321003，增三和弦
- Bbdim7 = x1202x，減七和弦（過渡用，很短）
- 主歌的 G→B7→Em7→Dm7→G 進行很好聽
- 間奏有 C#m7-5→Cmaj7→Bm7→Bbdim7 的半音下行
- 建議先練主歌+副歌，間奏/橋段熟了再加
