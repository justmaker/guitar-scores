## 練習筆記

- Em7 按法：020000（Em 放開無名指）
- Cadd9 按法：x32030
- D/F# 用拇指按第六弦第二格（參考 [和弦指法頁](../reference/chords.md)）
- Dsus4 按法：xx0233（D 加上小指按 1 弦第 3 格）
- B 是封閉和弦（x24442），初學可用 B7（x21202）替代
- 前奏是經典旋律，建議單獨練指彈
- 主歌用指法 T1 21 T1 21，副歌換刷弦加力度
- 注意副歌 G → Em7 → C → D 和 B → Em7 → C → Dsus4 D 的轉換
