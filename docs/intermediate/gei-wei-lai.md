```
前奏→主歌1→副歌1→間奏(G D/F# C Cm)
→主歌2→副歌2→副歌3→間奏2→
最終副歌→尾奏(G D/F# C Cm G)
```

## 練習筆記

- Cm 是封閉和弦（x35543），只出現在前奏/間奏很短
- Bm7 用簡易按法 x20202 就好
- B 封閉和弦出現在副歌（Am7 B | Em）
- 副歌的 C→D→Bm7→Em7→Am7 進行很順暢
- D/F# 用拇指按第六弦第二格
- 主歌指法 T1 21 T1 21，副歌換刷弦加力度
