## 練習筆記

- 原調 D，91 譜顯示 Capo 2 用 C 調指型
- 有 Bb 和 Fm 兩個封閉和弦，難度歸中級
- Bb 可用 A 型封閉第一格（x13331）
- Fm 是 Em 型封閉第一格（133111），出現在主歌過渡處很短
- Fm → G 的轉換是難點，可以先慢練
- 副歌的 F → C → Dm7 → G 進行很順，多練幾遍就熟
- 速度只有 64 BPM，很慢的抒情，適合練表情和力度控制
