## 練習筆記

- 這版和弦比較豐富（七和弦、sus4、轉位），歸類為中級
- D7、G7 是開放七和弦，按法簡單但色彩不同於普通大和弦
- G/B 低音走 B，讓 C → G/B → Am7 形成低音下行（很好聽）
- 預副歌 F → G → G/F → Em7 這段低音也是下行，注意 G/F 的轉換
- D7/F# 較難，可先用普通 D7 替代
- Gsus4 → G 是掛留解決，注意小指放開的時機
- 副歌 E7 只出現一次（第二段 G → E7），營造緊張感接 Am
