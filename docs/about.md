# 關於

這是我的吉他學習譜集，使用 MkDocs Material 建置，託管於 GitHub Pages。

## 如何新增一首歌

1. 在對應難度資料夾下建立 `.md` 檔
2. 使用下方模板
3. `git push` 後自動部署

## 譜的排版規則

所有譜頁遵守以下格式規範：

### 和弦行

- 每行**最多 4 格**和弦，每格約 5 字元寬
- 格式範例：`|C    |Am   |F    |G`
- 前奏/間奏純和弦行也遵守每行 4 格

### 歌詞行

- 歌詞緊接在和弦行**正下方**，在同一個 code block 裡
- 不使用大量空格對齊，保持自然閱讀

### 歌曲結構段

- 用簡寫，盡量一行內寫完
- 範例：`前奏→主歌1→副歌→間奏→主歌2→副歌→尾奏`

### 格式範例

```
|C    |C    |Em   |Em
 突如其來的美夢 是你離去時卷起的泡沫
|Am   |Am   |F    |F
 踢著石頭默默的走 公車從旁擦身而過
```
