# 關於

這是我的吉他學習譜集，使用 MkDocs Material 建置，託管於 GitHub Pages。

## 如何新增一首歌

1. 在對應難度資料夾下建立 `.md` 檔
2. 使用下方模板
3. `git push` 後自動部署

## 譜模板

```markdown
---
tags:
  - 難度-初級
  - 刷弦
---

# 歌名 - 歌手

| 資訊 | |
|------|------|
| 調性 | C |
| Capo | 0 |
| BPM | 120 |
| 拍號 | 4/4 |

## 和弦進行

\```
| C | G | Am | F |
\```

## 歌詞 + 和弦

\```
[C]歌詞在這裡  [G]和弦標在上方
[Am]繼續唱      [F]就這樣
\```

## 練習筆記

- 注意第二段的切音
- BPM 先從 80 開始練
```
