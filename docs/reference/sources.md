# 吉他譜來源網站

收錄我常用的吉他譜來源，方便查找。

## 主要來源

### 91譜（91pu）

- **網址**：[https://www.91pu.com.tw](https://www.91pu.com.tw)
- **特色**：
    - 華語歌曲齊全，譜量大
    - 支援線上轉調（自動移調）
    - 標示參考刷法與指法
    - 有和弦圖對照
    - 社群留言可參考勘誤
- **適合**：華語流行彈唱
- **備註**：本站所有譜以 91 譜為主要參考來源

### 搜尋方式

91 譜搜尋不是用 `search.php`，而是透過首頁表單提交。在瀏覽器中操作：

1. 開啟 `https://www.91pu.com.tw`
2. 在「請輸入關鍵字」搜尋框輸入歌名或歌手
3. 點擊 GO 按鈕
4. 結果頁會顯示在同一頁面

## 其他來源

### 譜全了（guistudy.com）

- **網址**：[https://guistudy.com](https://guistudy.com)
- **特色**：
    - 海量曲譜免費下載
    - 支援吉他譜 + 烏克麗麗譜
    - 有彈唱譜、指彈譜、和弦譜多種格式
    - 高清圖片譜，適合在手機/平板看
    - 中文歌曲豐富（周杰倫、趙雷、林俊傑等）
- **適合**：想找高清圖片譜、或 91 譜沒收錄的歌曲時的替代來源
