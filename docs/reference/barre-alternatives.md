# 封閉和弦替代方案

初學遇到封閉和弦（Barre Chords）不用怕！每個封閉和弦都有開放和弦的替代按法，聲音接近、功能相同，讓你先把歌彈順再慢慢練封閉。

## B

詳見 [B 和弦攻略](b-chord.md)

| 按法 | 難度 | 說明 |
|------|------|------|
| B7（x21202） | ⭐ | 最簡單替代 |
| 迷你 B（x244xx） | ⭐⭐ | 只按 3 弦 |
| 半封閉 B（x2444x） | ⭐⭐ | 封 4 弦 |
| 完整 B（x24442） | ⭐⭐⭐ | A 型封閉 |

## 練習建議

1. **先用替代和弦把歌彈順**，不要卡在一個和弦上讓整首歌停下來
2. **每天花 5 分鐘專練封閉**：食指平壓練習，從第 5 格開始（弦張力較低）
3. **漸進式**：替代 → 迷你版 → 半封閉 → 完整封閉
4. **不要硬練到痛**：痛了就休息，肌肉是慢慢長出來的
