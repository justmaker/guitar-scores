# 封閉和弦替代方案

初學遇到封閉和弦（Barre Chords）不用怕！每個封閉和弦都有開放和弦的替代按法，聲音接近、功能相同，讓你先把歌彈順再慢慢練封閉。

## B

| 按法 | 難度 | 說明 |
|------|------|------|
| B7（x21202） | ⭐ | 最簡單替代，七和弦色彩稍不同但功能一樣 |
| Badd11（x2440x） | ⭐⭐ | 只按中間四弦，聲音較現代 |
| B（x244xx） | ⭐⭐ | 只按 A D G 三弦的迷你版 |
| 完整 B（x24442） | ⭐⭐⭐ | A 型封閉，最終目標 |

**B7 按法：**
```
e ─── 2 ─── ← 中指
B ─── 0 ───
G ─── 2 ─── ← 無名指
D ─── 1 ─── ← 食指
A ─── 2 ─── ← 小指
E ─── X ───
```

## 練習建議

1. **先用替代和弦把歌彈順**，不要卡在一個和弦上讓整首歌停下來
2. **每天花 5 分鐘專練封閉**：食指平壓練習，從第 5 格開始（弦張力較低）
3. **漸進式**：迷你版 → 半封閉 → 完整封閉
4. **手指位置**：食指用側面、靠近琴衍、拇指在琴頸正後方
5. **不要硬練到痛**：痛了就休息，肌肉是慢慢長出來的
