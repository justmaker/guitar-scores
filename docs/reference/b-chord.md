# B 和弦攻略

B 是繼 F 之後初學者第二個大魔王，因為在第 2 格做 A 型封閉。

## 1. B7（入門替代）

`x21202`

![B7](img/b7_open.png)

!!! tip "適用場景"
    開放和弦，不用封閉。七和弦色彩略有不同但功能完全相同。

## 2. 迷你 B（只按 3 弦）

`x244xx`

![迷你 B](img/b_mini.png)

!!! tip "適用場景"
    只按 A D G 三弦，聲音薄但確實是 B 大和弦。

## 3. 半封閉 B（封 4 弦）

`x2444x`

![半封閉 B](img/b_half.png)

!!! tip "適用場景"
    食指只壓 A 弦第 2 格，其餘分別按第 4 格。聲音已飽滿。

## 4. 完整大封閉 B（最終目標）

`x24442`

![B](img/b.png)

## 練習路線

```
B7 → 迷你 B → 半封閉 B → 大封閉 B
```

1. **先用 B7 把歌彈順**，不要卡住讓練習中斷
2. **迷你 B 練手指展開**：食指第 2 格、其餘第 4 格的距離感
3. **半封閉加入更多弦**：練習三指同時按第 4 格
4. **大封閉是時間問題**：跟 F 一樣慢慢來

!!! warning "常見錯誤"
    - 中指、無名指、小指沒有立起來 → 會悶到旁邊的弦
    - 食指封閉時離琴衍太遠 → 靠近第 2 格金屬條省力
    - 可以用無名指一根橫壓 D G B 三弦（進階技巧）
    - 不要跟 F 一起練 → 一天專注一個封閉和弦就好

## B vs F 差異

| | F（E 型封閉） | B（A 型封閉） |
|---|---|---|
| 位置 | 第 1 格 | 第 2 格 |
| 封閉弦數 | 6 弦全封 | 5 弦 |
| 難點 | 食指壓滿 6 弦 | 中間三指擠同格 |
| 替代 | Fmaj7 | B7 |
