# 烏克麗麗和弦指法

烏克麗麗只有 4 弦（G C E A），大部分吉他和弦都有對應。

> **弦序（粗→細）：** G C E A
>
> 指法字串從 G 弦到 A 弦，例如 `0003` = G空 C空 E空 A3格
>
> **轉位和弦規則：** 遇到 slash 和弦（如 D/F#、G/B）直接忽略 "/" 後面的低音，彈基本和弦即可。烏克沒有低音弦，差異極小。

> **手指代號：** ① 食指 ② 中指 ③ 無名指 ④ 小指
