## 轉位和弦簡化對照

吉他上有些轉位/slash 和弦在烏克上不存在（因為少了低音弦），直接用基本形式彈即可：

| 吉他和弦 | 烏克簡化為 | 烏克指法 | 原因 |
|----------|-----------|----------|------|
| D/F# | D | 2220 | 烏克沒有低音 F# 弦 |
| G/B | G | 0232 | 烏克沒有低音 B 弦 |
| G/F | G7 | 0212 | 用 G7 代替（含 F 音） |
| C/E | C | 0003 | 烏克沒有低音 E 弦 |
| Bm/D | Bm | 4222 | |
| B7/Eb | B7 | 2322 | |
| B/Eb | B | 4322 | |
| G/D | G | 0232 | |

## 練習建議

1. **烏克比吉他簡單很多** — 大部分和弦只要 1-2 根手指
2. **Am7 = 全空弦** — 烏克最簡單的和弦
3. **轉位和弦不用管** — 直接彈基本形式，聲音差異很小
4. **封閉和弦少** — 只有 Bm、B、Bb 等少數需要封閉
5. **本站的譜都可以用烏克彈** — 照上面對照表轉換就好

## 指法圖說明

```
烏克弦序（從左到右）：G C E A
數字：0=空弦 1-N=第幾格
例：C = 0003 表示只按 A 弦第 3 格
```
