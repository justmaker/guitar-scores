## 轉位和弦簡化對照

吉他上的轉位/slash 和弦在烏克上直接用基本形式彈：

| 吉他和弦 | 烏克簡化為 | 指法 |
|----------|-----------|------|
| D/F# | D | 2220 |
| G/B | G | 0232 |
| G/F | G7 | 0212 |
| C/E | C | 0003 |
| Bm/D | Bm | 4222 |
| B7/Eb | B7 | 2322 |
| B/Eb | B | 4322 |
| G/D | G | 0232 |

## 指法圖說明

```
弦序（從左到右）：G C E A
數字：0=空弦 1-N=第幾格
例：C = 0003 → 只按 A 弦第 3 格
```

## 練習建議

- **Am7 = 全空弦** — 烏克最簡單的和弦
- **轉位和弦不用管** — 直接彈基本形式
- **本站所有譜都可以用烏克彈** — 照上面對照表轉換
