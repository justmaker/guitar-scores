
## 練習筆記

- 全曲只用 Em, C, G, D 四個開放和弦，無敵簡單
- 和弦進行從頭到尾都是 Em → C → G → D 循環
- Capo 2 彈 Em 調 = 原調 F#m
- 刷法 X ↑ ↑↓ X ↑ ↑↓ 有律動感，注意悶音（X）要乾淨
- 也可以用指法 T1 21 T1 21 來彈前奏和主歌
- 這首非常適合當第一首練習曲，因為和弦轉換只有一種模式
- 重點在於節奏穩定和悶音技巧
