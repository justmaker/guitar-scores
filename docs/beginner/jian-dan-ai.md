## 練習筆記

- 主歌 C → G → Am → F 是經典四和弦，初學友善
- 預副歌有 Dm7、G/B 的低音下行，增加層次感
- Bb 只出現一次（預副歌最後），可先用 F 替代簡化
- 刷法有「打板」技巧（T1 打1），用手掌輕拍面板製造節奏
- BPM 97 速度中等偏快，建議先從 80 開始
- 副歌 C → G/B → Am → Em7 低音下行很好聽，注意 G/B 的轉換
