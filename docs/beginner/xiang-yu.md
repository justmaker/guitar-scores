[G](../reference/chords.md#g), [Em7](../reference/chords.md#em7), Cmaj7, [Dsus4](../reference/chords.md#dsus4), [D](../reference/chords.md#d)

```
前奏→主歌1→副歌→

```

## 練習筆記

- 全曲只用 G, Em7, Cmaj7, Dsus4, D 五個和弦，皆為開放指型
- Cmaj7 = C 和弦但不壓第一弦（X32000）
- Em7 = Em 放開無名指
- Dsus4 = D 加上小指壓第一弦第三格
- BPM 71 偏慢，適合初學者
