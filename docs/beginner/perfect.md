## 使用和弦

[G](../reference/chords.md#g), [Em](../reference/chords.md#em), [C](../reference/chords.md#c), [D](../reference/chords.md#d)

## 刷法與指法

```
6/8 拍指法：T 3 2 1 2 3（每拍一個音，一小節六個音）
或簡化刷法：↓ · · ↓↑ ·（慢華爾滋感）
```

## 練習筆記

- 全曲只用 G, Em, C, D 四個開放和弦，非常適合初學
- 6/8 拍（華爾滋感），每小節數 1-2-3-4-5-6
- 右手指法推薦 T 3 2 1 2 3，營造流動感
- 也可以用 T 3 21 3 2（稍加變化）
- Capo 1 彈 G 調 = 原調 Ab
- 速度很慢（63 BPM），適合練分解和弦的穩定度
- 重點：讓每個音都飽滿乾淨，不要趕拍
